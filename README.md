## 🦠 TASK 3: COVID-19 Data Report (Status: Completed ✅)

### 📌 Project Objective
To perform a structured time-series analysis and epidemiological tracking using global COVID-19 chronological datasets. This project focuses on mutating data objects into proper datetime vectors, isolating national tracking filters, computing 7-day moving averages to smooth daily reporting anomalies, and performing comparative trajectory plotting across multiple global regions.

### 📂 Files in Repository for Task 3:
* `covid_19_clean_complete.csv` : Time-series source data used for chronological engineering.
* `Cognetix_Covid19_Data_Report.ipynb` : Complete Python source code notebook with trend plots.
* `Cognetix_Covid19_Data_Report.pdf` : Executive PDF report mapping the structured outputs.
